Data Flow:
1. User interacts with View (e.g., clicks "Start Trading")
2. View calls method on State object (e.g., AppState.startTrading())
3. State object calls APIService method
4. APIService makes HTTP request to backend
5. Backend processes and responds
6. APIService decodes response
7. State object updates @Published properties
8. SwiftUI automatically refreshes affected Views

WebSocket Flow:
1. ConnectionState.connect() establishes WebSocket
2. Backend sends real-time updates
3. ConnectionState.receiveMessage() parses JSON
4. Updates forwarded to relevant State objects
5. Views automatically refresh with new data
